# Discord AI Bot — n8n + Groq

This project implements an intelligent Discord bot that leverages n8n workflow automation and Groq's powerful LLaMA 3.3 70B language model to provide instant AI-powered responses to user queries in Discord channels. The bot listens for messages prefixed with "!ask" and responds with helpful, concise answers generated by advanced AI.

## Architecture

```
Discord Message → n8n (polls every 10s) → Groq LLaMA 3.3 70B → Discord Reply
```

## Tech Stack

- **n8n**: Workflow automation platform for orchestrating the bot logic
- **Groq**: High-performance AI inference with LLaMA 3.3 70B model
- **Discord API**: For reading messages and posting responses

## How It Works

1. **Polling**: n8n triggers every 10 seconds to check for new messages in the specified Discord channel
2. **Message Detection**: Filters messages that start with "!ask" command
3. **Question Extraction**: Parses the user question from the message content
4. **AI Processing**: Sends the question to Groq's LLaMA 3.3 70B model for intelligent response generation
5. **Response Delivery**: Posts the AI-generated answer back to Discord using a webhook

## Setup Instructions

1. **Prerequisites**:
   - n8n instance (self-hosted or cloud)
   - Discord Bot application
   - Groq API account

2. **Discord Setup**:
   - Create a Discord application at https://discord.com/developers/applications
   - Add a bot to your application
   - Copy the bot token
   - Create a webhook in your Discord server/channel

3. **Groq Setup**:
   - Sign up at https://groq.com
   - Generate an API key

4. **n8n Configuration**:
   - Import the `Discord Ai Bot.json` workflow file
   - Set up credentials for Discord Bot Token and Groq API Key
   - Update the webhook URL and channel ID in the workflow nodes
   - Activate the workflow

5. **Environment Variables**:
   Create a `.env` file based on `.env.example` and fill in your actual credentials.

## Environment Variables

| Variable | Description | Where to Get It |
|----------|-------------|-----------------|
| `GROQ_API_KEY` | Your Groq API key for authentication | https://console.groq.com/keys |
| `DISCORD_BOT_TOKEN` | Discord bot token for API access | https://discord.com/developers/applications |
| `DISCORD_WEBHOOK_URL` | Full webhook URL for posting responses | Discord server settings → Integrations → Webhooks |
| `DISCORD_CHANNEL_ID` | The Discord channel ID to monitor | Right-click channel → Copy ID (requires Developer Mode) |
